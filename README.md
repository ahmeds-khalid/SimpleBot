## Setup

1. **Requirements**
   - Python 3.8+
   - Discord Bot Token

2. **Installation**
*Click on the green button "Code", then click "Downnload Zip" and unzip the file*

3. **Configuration**
   - Create `.env` file:
     ```
     DISCORD_TOKEN=your_bot_token_here
     ```

4. **Discord Setup**
   - Create bot in [Discord Developer Portal](https://discord.com/developers/applications)
   - Enable Message Content & Server Members Intents
   - Invite bot to your server

5. **Run**
```bash
python main.py
```
